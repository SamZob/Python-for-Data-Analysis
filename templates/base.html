Page Block Classification

The problem consists in classifying all the blocks of the page layout of a document that has been detected by a segmentation process.

This is an essential step in document analysis in order to separate text from graphic areas. There are five classes of page blocks:

  - Text (1)

  - Horizontal line (2)

  - Graphic (3)

  - Vertical line (4)

  - Picture (5).
